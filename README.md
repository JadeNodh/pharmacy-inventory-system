# Pharmacy & Inventory Control System

## Description
A Laravel-based web application to manage medicines and inventory with full CRUD operations and reporting features.

## Features
- Medicine Management (CRUD)
- Inventory Management (CRUD)
- Reports:
  - Medicine List Report
  - Expiry Medicines Report
  - Low Stock Report
  - Inventory Stock Report
  - Search Report
- Form Validation

## Technologies Used
- Laravel
- MySQL
- Bootstrap

## Setup Instructions
1. Clone the repository
2. Run `composer install`
3. Configure `.env`
4. Run `php artisan migrate`
5. Run `php artisan serve`
